Dicas

Comentários

Para acresentar um comentário em HTML vamos usar o comando <!-- -->e tudo o que tiver dentro disso se torna um comentário.

Envelopar

Para envelopar basta seguir os passos:

  1. Selecionar onde deseja envelopar
  2. Aperte Shift + Ctrl + P
  3. Selecione a opção: Emmet: Wrap with Abbreviation
  4. Digite e tag desejada
  5. Aperte enter para confirmar

Mais de cursor

Para adicionar cursor basta clicar onde desejar, segure Alt + Clique onde quiser o outro cursor.

Endentação

Para afastar a identação para a direita use a tecla Tab
Para afastar a identação para a esquerda use a tecla Shift + Tab

Listas

Para criar uma lista com vários itens utilize o comando ul > li * (quantidade de itens na lista)
Ex: Para criar uma lista com 5 itens - ul > li * 5

Criando div já com uma classe

Para agilizar e criar uma div já com uma classe use o comando div.(nome da classe)

Renomear Tags

Para renomear de uma fomra mais fácil, basta clicar na tag com o botão direito e clicar em renomear, vai abrir uma caixa de texto e depois so digitar a nova tag e apertar enter.

Links

Para criar um link usamos a tag a e dentro dela o href para indicar o endereço em que vamos ser direcionado. Porém, se usarmos o caractere # e logo depois adiconarmos o id que queremos, podemos criar um link para a mesma página, como um portifólio.

Link para Email

Para criar um link onde quando o usuário clicar, o programa de email dele abre já com o destinatário de email preenchido.
