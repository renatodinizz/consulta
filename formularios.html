Formulários

Criando um formulário simples.

Para se criar um formulário como esse exemplo usamos o comando form.
Inserimos um parágrafo onde no exemplo foi escrito nome.
Depois inserimos o comando input:text para criar essa caixa, onde o texto pode ser escrito.
Para criar o botão usamos o comando input:submit e dentro do parametro value, colocamos o que vai estar escrito dentro do botão.

Podemos tirar o auto completar usando o parametro autocomplete='off' dentro da tag form.

O parametro action serve para colocar o endereço onde os dados vão ser salvos, logo depois os dados serão enviados quando o botão de submit for utilizado.
