Formulários

Criando um formulário simples.

Para criar um formulário como esse exemplo usamos o comando form.
Inserimos um parágrafo onde no exemplo foi escrito nome.
Depois inserimos o comando input:text para criar essa caixa, onde o texto pode ser escrito.
Para criar o botão usamos o comando input:submit e dentro do parametro value, colocamos o que vai estar escrito dentro do botão.

Podemos tirar o auto completar usando o parametro autocomplete='off' dentro da tag form.

O parametro action serve para colocar o endereço onde os dados serão salvos, logo depois os dados serão enviados quando o botão de submit for utilizado.

Métodos GET e POST

Podemos utilizar os métodos GET ou POST para formulários.

GET - É o método padrão para formelários, é utilizado para campos simples como cadastros, peso, altura, buscas e informações que não sejam sensíveis.
O limite para uso do GET são de 3 mil bites, aproximadamente 3 mil palavras.

POST - Para utilizar o método POST é necessário fazer a alteração no parametro do form, esse método é recomendado a ser utilizado para senhas, número do cartão de crédito, endereços, envio de arquivos e etc.

Tipos de Input

Input:Password - Utilizado para campos de senhas, os caracteres do campo digitado não são visivéis.

Input:Reset - Utilizado para limpar os campos digitados.

Input:Numer - Utilizado para campos onde não aceita letras, apenas números.

Input:Month - Utilizado para campos onde só se aceita meses e anos.

Input:Date - Utilizado para campos onde só se aceita dia, meses e anos.

Input:Time - Utilizado para campos onde só se horas e minutos.

Tipos de parametros

required - Classifica como obrigatório o campo a ser preenchido.

minlength - Limita o número mínimo de caracteres a ser preenchido.

maxlength - Limita o número máximo de caracteres a ser preenchido.

size - Limita o número de caracteres a ser visualizado no campo preenchido.

placeholder - Mostra uma dica no campo a ser preenchido. ex.(min. 8 caraxteres)

autocomplete - Dentro da tag input serve para relacionar o que cada input seria de fato, assim o navegador vai relacionar que aquele campo é para um nome de usuário, senha ou etc. Isso serve para que o usuário peça ao navegador que preencha automaticamente os campos do formulário.
Obs: No parametro autocomplete, existe diversos parametros, como username, new-password, current-password

min - Limita o número mínimo de números a ser preenchido.

max - Limita o número máximo de números a ser preenchido.

step - Limita o quanto um número pode variar ex.(step'0.5') vai aceitar somente números de 0.5 e assim por diante.

value - Sugere um valor para o campo a ser preenchido.
