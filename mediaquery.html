Media Query

Media Types

Para uma media type em CSS, precisamos de um link:css para fazer a configuração necessária e utilizamos o parametro media, que indica o tipo de media query vai ser utilizada na CSS.
Desta forma.

Media Features

A função principal das media features é personalizar as configurações para cada tipo de tela ou função, como exemplo uma tela grande para uma tela de celular ou uma página de impressão.

Nesse exmeplo temos um link:css all que personaliza de forma geral, para não repetir código em outras folhas de CSS.
link:css portrait - Que personaliza a página que ficar em modo retrato.
link:css landscape - Que personaliza a página que ficar em modo paisagem.

Transição

Para criar uma animação em uma transição de uma tela para outra,usamos o comando transition: width 0.3s, height: 0.3s, o resultado é uma animação interessante.

transition-timing-function: ease; - Faz com que a animação seja mais suave.
