DOM - Document Object Model

Árvore DOM

Vamos analisar um exemplo de uma árvore DOM

Selecionando elementos

Para selecinar elementos, podemos selecinar por:
Marca
ID
Nome
Classe
Seletor

Selecionar por marca

Para selecionar por marca, usamos o comando: window.document.getElementsByTagName('p')[1]
Neste caso, o comando indica que o elemento a ser selecionado é uma tag.
Dentro do paragráfo está especidficando qual tag será selecionada
O colchetes está indicando qual dentre os paragráfos será selecionado.

Selecionar por ID

Para selecionar por ID, usamos o comando: window.document.getElementById()

Selecionar por nome

Para selecionar por nome, usamos o comando: window.document.getElementsByName()[]

Selecionar por classe

Para selecionar por classe, usamos o comando: window.document.getElementsByClassName()[]

Selecionar por seletor

Para selecionar por seletor, usamos o comando:
window.document.querySelector('div.tag') ou ('id#tag')
ou o comando:
window.document.querySelectorAll('div.tag') ou ('id#tag')

Alguns outros comandos com DOM

  • p1.style.color - Muda a cor da variável.
  • document.body.style.background - Muda a cor da de fundo.
