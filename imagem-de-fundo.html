Imagem de fundo

Inserindo imagem de fundo

  • Utiliza-se o comando background-image
  • Com o parametro Url
  • E coloca o caminho da imagem desejada
  • Ps: É recomendado sempre colocar uma cor no fundo mesmo quando for inserir uma imagem de fundo

Posicionamento da origem de inicio da imagem

Para posicionar o ponto de partida da imagem, podemos imaginar 3 colunas e 3 linhas
As colunas: Left, center e right
linhas: Top, center e bottom

O comando a ser utilizado é o background-position
Ex: background-position: right center;

Outros comandos para estilização

Por padrão a imagem de fundo será repitida quando a tela for grande e para corrigir isso vamos usar o comando background-repeat.
Vamos usar os parametros a seguir para personalizar da melhor forma.

  • repeat - Que já é o padrão
  • no-repeat - Para não repetir
  • repeat-y - Para repetir somente na coluna y
  • repeat-x - Para repetir somente na lina x

Para o tamanho utiliza-se o comando background-size e vamos ver alguns parametros para estilizar.

  • 800px - Pode ser usado o tamanho que desejar
  • 100% - Vai ocupar toda a tela se a imagem for na dimensão da tela e a tela não tiver rolagem
  • contain - Faz a imagem ser 100% exibida mas deixa aquelas barras pretas para ajustar a imagem.
  • cover - Faz a imagem ser 100% exibida mesmo que para isso ele tenha que corta um pouco a imagem.

Para ajustar a imagem de fundo utiliza-se o comando background-attachment com os parametros

  • scroll - Para a imagem rolar junto com o conteúdo
  • fixed - Para a imagem permanecer fixa.

Short hand

Para a short hand podemos usar o comando background e inserir os valores na ordem a seguir.

  • color
  • imagem
  • position
  • repeat
  • attachment

Ex: background: black url('imagem/wallpaper002.jpg') center center no-repeat fixed;
Obs: Não é possivél usar o background-size no short hand, deve se usar separado.
