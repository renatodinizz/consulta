Criando funções

Funções são ações executadas assim que são chamadas ou em decorrência de algum evento.

Uma função em JavaScript é criada usando a palavra reservada function

Então para criar uma função, usamos o seguinte comando: 

                function NOME DA FUNÇÃO (PARAMETROS) {
                    Dentro das chaves o comando do que fazer.
                }

Alguns comandos com o mouse

Lmebrar sempre de colocar a preposição on antes dos comando de mouse.

  • mouseenter - Interage quando o mouse entrar no conteúdo.
  • mouseout - Interage quando o mouse sair do conteúdo.
  • mousemove - Interage quando o mouse mover dentro do conteúdo.
  • mousedown - Interage quando o usuário clica e segura o mouse no conteúdo.
  • mouseup - Interage quando o usuário soltar o mouse que tinha sido clicado e segurado.
  • click - Interage quando o mouse clica no conteúdo.
