Trabalhando com edição de textos

Títulos

Podemos inserir títulos com o comando H + algum número!

A ordem dos títulos vão do <h1> ao <h6>

Parágrafos

Para criar um parágrafo basta usar o comando <p>

Pode-se criar um parágrafo de exemplo usando o comando "lorem"
ex: Lorem ipsum dolor sit amet consectetur, adipisicing elit. Odio libero, impedit ducimus voluptate repellendus nemo corrupti velit reprehenderit. Unde vel reprehenderit, ab accusamus temporibus expedita commodi consequuntur ea numquam voluptas.

Para quebrar uma linha utilize o comando <br>

Para inserir uma linha horizontal utilize o comando <hr>

Negrito

Com o comando <Strong> o texto fica em negrito - Desta forma

Itálico

Com o comando <em> o texto fica em itálico - Desta forma

Marcação

Com o comando <Mark> o texto fica marcado - Desta forma
Com CSS você pode escolher a cor da marcação.

Diminuir fonte

Com o comando <Small> o texto fica pequeno - Desta forma

Riscar o texto

Com o comando <Del> o texto fica ricado - Desta forma

Subilinhar texto

Com o comando <Ins> o texto fica subilinhado - Desta forma

Texto invertido

Com o comando <bdo> o texto fica invertido - Desta forma
Precisa colocar o comando rtl no campo dir para funcionar.

Sobrescrever texto

Com o comando <Sup> o texto fica sobrescrito - ex: X9+32=100
Aqui o número 9 está sobrescrito.

Subescrever texto

Com o comando <Sub> o texto fica subescrito - ex: H2O
Aqui o número 2 está subescrito.

Monoespaçar texto

Com o comando <Code> o texto fica monoespaçado.
Todos os caracteries irão ter o mesmo espaço.

Exemplo de um texto sem monoespaçamento: document.getElement.id

Exemplo de um texto com monoespaçamento: document.getElement.id

Não sei o que aconteceu mas não ficou claro o comando, acho que foi a fonte que mudou!

Pre formatar como código

Com o comando <Pre> o texto fica pre formatado como no exemplo abaixo

Exemplo de texto sem a tag PRE

num = int(imput("digite um número")) if num % 2 == 0: print(f "o número(num) é PAR") else: print(f "o número(num) é IMPAR") print("fim do projeto")

Exemplo de texto com a tag PRE

            
                num = int(imput("digite um número"))
                if num % 2 == 0:
                    print(f "o número(num) é PAR")
                else:
                    print(f "o número(num) é IMPAR")
                print("fim do projeto")

Citações simples

Com o comando <Q> o texto fica como uma citação.
Exemplo: O pai de um amigo meu já dizia: O computador é um burro muito rápido

Citações complexas

Com o comando <blockcode> faz referência a uma grande citação. Dentro do comando existe um campo para colocar um link, o que ajuda os buscadores a saberem a referência que está sendo usada.

Abreviações

Com o comando <Abbr> exibe uma mensagem quando o cursor fica parado por cima da palavra.

Exemplo: Estou estudando HTML e CSS, estou adorando!

Lista Ordenada

Com o comando <ol> podemos criar uma lista ordenada.
Cada item da lista deve ser usada dentro do comando <li>
Pode usar diferentes tipos de caracteres como: 1, a, A, i, I
Pode usar no comando start em que posição deseja começar.
Exemplo:

  1. levantar
  2. escovar os dentes
  3. vestir uma roupa
  4. Ir para academia
  5. Tomar banho
  6. Estudar

Lista Desordenada

Com o comando <ul> podemos criar uma lista desordenada.
Cada item da lista deve ser usada dentro do comando <li>
Pode usar diferentes tipos de marcadores como: disc, circle, square
Exemplo:

  • Estudar
  • Treinar
  • Jogar basquete
  • Jiu-Jitsu
  • Relaxar
