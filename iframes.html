Inserindo um Iframe

Para adicionar um iframe basta usar o comando Iframe, como no exemplo abaixo.

Acessando o site do para aprender a programar.
Para configura o iframe basta usar CSS. Por padrão o iframe vem com a medida 300 por 150 px.

Podemos usar alguns parametros dentro do html para configurar o iframe, como o scolling onde podemos configurar...
O modo auto para se o conteúdo esceder o padrão do frame ele insere uma barra de rolagem;
O modo yes para exibir sempre a barra de rolagem;
O modo no para não exibir uma barra de rolagem, mesmo que o conteúdo não caiba no frame.
