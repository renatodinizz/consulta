Responsividade de imagens

Para deixar um site responsivo, isso é, adaptado a diferentes tamanhos, sigua os passos abaixo:

  • Em CSS na tag main use os comandos abaixo
  • min-width: 320px; (suguerido usar esse tamanho)
  • max-width: 800px; (suguerido usar esse tamanho)

Assim to texto irá fica responsivo, para as imagens siga as instruções abaixo:

  • Em CSS chame a imagem desejado e coloque o comando width: 100%;
  • Crie uma tag Picture
  • Dentro da tag Picture insira uma imagem com o comando img
  • Na linha em cima da imagem use o comando Source:media
  • Mude o paramêtro de min-width para max-width e coloque a largura desejada
  • Insira manualmente a outra imagem que irá fazer a responsividade

Responsividade com videos

É difícil de explicar mas qualquer coisa, dê uma olhada no código, mas vou me esforçar ao máximo para deixar bem claro.
Para deixar um video responsivo siga os passos abaixo:

  • Insira o video em uma div e dê uma classe para ele
  • Em CSS crie um comando para chamar a div
  • Insira o comando position: relative;
  • Insira um padding-bottom: 59% (valor testado)
    Obs: O vídeo vai ficar com barras pretas nas laterias isso pode ser ajustado com o comando paddin-bottom
  • Crie um comando chamando o iframe do video
  • Ex: div.video > iframe{}
  • Insira o comando position: absolute;
  • Insira o comando top: 5%;
  • Insira o comando left: 5%;
  • Insira o comando width: 90%; (Só é possivél pois estsmao trabalhando em top e left com porcentagem)
  • Insira o comando height: 90%; (Só é possivél pois estsmao trabalhando em top e left com porcentagem)
