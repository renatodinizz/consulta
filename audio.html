Trabalhando com audios

Inserir audio

Para inserir audio usamos o comando <audio>
Dentro da tag audio insira o comando controls para poder controlar o arquivo dentro da página.
Os formatos aceitos pelos navegadores são MP3, WAV e OGG

Inserir vários formatos

Aprendendo a inserir audio parecido com o comando IF em JavaScript.

Usamos o comando <audio> e do comando <audio> usamos o comando <source:src>
Obs: Usa-se o type audio/mpeg para aquivos em MP3
O parâmetro Preload éa forma como o arquivo vai carregar na página (recomanda usar o parâmetro metadata)
Pode-se usar o parâmetro loop para repetir o arquivo
