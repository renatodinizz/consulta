Estilizar com CSS

Seletores

Para configurar as CSS podemos usar varios seletores, que é basicamente a maneira como irá chamar o elemento do HTML na CSS, temos os seletores como:

  • Seletor global - Usado para todos os elementos no projeto.
    Ex: *
  • Seletor de tags - Usado para selecionar elementos específicos.
    Ex: h1
  • Seletor de elementos aninhados - Usado para selecionar elementos que estão dentro de outros elementos.
    Ex: nav a
  • Seletor de filhos - Usado para selecionar elementos específicos que estão dentro de outro elemento.
    Ex: nav > a
  • Seletor de classes - Usado para selecionar elementos que estão dentro de uma classe especifica.
    Ex: .nome_da_calsse
  • Seletor de id - Usado para selecionar elementos que estão identificados com um id.
    Ex: #nome_do_id
  • Combinação de seletor * - Usado para selecionar todos os elementos que estão dentro de outro elemento.
    Ex: secao-pricipal > *
  • Seletor de sequência - Usado para selecionar um elemento que está depois de outro, no caso do exemplo somente o segundo elementos será selecionado.
    Ex: h1 + p
  • Seletor de propriedade - Usado para selecionar um elemento com uma propriedade específica como name, type e etc.
    Ex: input[type="password"]
  • Seletor de pseudo class - Usado para selecionar uma pseudo classe.
    Ex: button:hover

Seletores Avançado

Podemos usar alguns seletores de forma mais avançadas, isso serve principalmente para quando não se tem acesso ao HTML da aplicação como no caso de usar uma ferramenta como WordPress ou outra parecida.

Para selecionar um elemento usamos um seletor avançado, vamos imaginar que queremos selecionar um elemento li que está em uma lista, para esse exemplo vamos usar o comando:
Ex: li:nth-child(3){} - Nesse exemplo o li é o item a ser selecionado, nth vem de: number e th da ordem cardial em inglês, child: por que é um filho do elemento ul e o número dentro do parênteses está indicando que dentro da ul o terceiro li será selecionado.
Obs: Nesse caso consideramos o primeiro intem como sendo o número 1.

Ex: li:nth-child(2n){} - Nesse exemplo o n adicional depois do número significa que ele vai pular de dois em dois intem, ou seja, somente números pares, mas podemos substiruir essa expressão pela palavra even dentro dos parênteses.
Ex: li:nth-child(2n-1){} - Nesse exemplo o n adicional depois do número e a expressão significa que ele vai pular de dois em dois intem, mas começando pelo número 2-1 que é 1, ou seja, irá contar somente os números impares, mas podemos substiruir essa expressão pela palavra odd dentro dos parênteses.

Ex: li:first-child - Seleciona o primeiro elemento filho.
Ex: li:last-child - Seleciona o último elemento filho.
Ex: li:not(first-child) - Seleciona todos os elementos menos o primeiro.

Pseudos Classe

:focus: - Atribui ações quando o elemento recebe um foco, como por exemplo, quando há um clique em uma caixa de input.

:hover: - Atribui ações quando o mouse passa por cima do elemento.

Pseudos Elementos

:: before{contete:"o que irá ser adicionado" ;} - Atribui um texto, imagem ou símbolo antes do elementos selecionado.

:: after{contete:"o que irá ser adicionado" ;} - Atribui um texto, imagem ou símbolo depois do elementos selecionado.

Alguns comandos

Comandos que eu acho necessário

  • padding - Controlam as margens
  • border-radius - Arredonda as margens
  • box-shadow - Constroi uma sombra nas bordas Obs: se colocar os dois primeiros valores zerados a sombra fica dos dois lado
  • text-shadow - Constroi uma sombra no texto
  • text-indent - Para adicionar um espaço no parágrafo
  • font-style: italic - Paracolocar em itálico
  • font-decoration: underline - Para subilinhar um texto
  • margin:auto - Para centralizar uma caixa
  • Transition-duration - Para evidenciar quando o mouse passar em cima
    Ex: transition-duration: 0.5s;
  • line-hight - Para mudar o tamnho do espaçmento entre uma linha e outra
  • text-transform: uppercase; - Para colocar todas as letras em maiusculas
  • font-variant: small-caps; - Para colocar todas as letras em maiusculas, dando destaque para as primeiras letras de cada palavra
  • Cursor: pointer - O ícone do mouse vira uma mãozinha
  • Overflow: hidden - Não deixa o conteúdo passar do limeite
  • letter-spacing: Atribui um espaçamento para todas as letras.
  • box-sizing: É a forma como o tamanho de um elemento vai ser calculado, usado muito em seletor global.
  • box-sizing: border-box - Vai calcular o tamanho de acordo com o conteúdo até a borda, ou seja, conteúdo + padding + borda.
  • box-sizing: content-box - Vai calcular o tamanho de acordo com o conteúdo. É o valor padrão.

Unidades de medidas

  • Pixel (px) - VUnidade fixa, não tem alteração.
    Ex: 30px - Será 30px independente do tamanho da tela.
  • Root Element (rem) - Vai alterar conforme o tamanho da raiz do elemento.
    Ex: 3rem - Vai aumentar 3x o tamanho do elemento.
  • Em (em) - Vai alterar conforme o tamanho do elemento atual.
    Ex: 3rem - Vai aumentar 3x o tamanho do elemento.
  • Porcentagem (%) - Vai alterar percentualmente conforme o tamanho do elemento pai.
    Ex: 30% - Vai aumentar 30% o tamanho do elemento pai.
  • ViewPort Heigh (vh) - Vai alterar a altura conforme o tamanho da tela.
    Ex: 30vh - Vai dimencionar em 30% a altura da tela, sendo ela pequena ou grande.
  • ViewPort Width (vw) - Vai alterar a largura conforme o tamanho da tela.
    Ex: 50vh - Vai dimencionar em 50% a largura da tela, sendo ela pequena ou grande.

Degradê

Para acrescentar um degradê use o comando background-image: linear-gradient()
Obs: Dentro dos parênteses indique a direção do degradê como: to right, to left, to top, to bottom ou por graus como: 90deg, 45deg
Obs: Depois da direção informe as cores que irão compor o degradê.

Algumas configurações para o degradê:

  • Para fixar o degradê - background-attachment: fixed;
  • Para fazer um degradê circular - radial-gradient: circle; (no lugar de linear-gradient).
  • Para dimensionar quanto de cada cor - Basta coloar a quantidade em porcentagem depois da cor.

Importando fontes

Para importar uma fonte, basta seguir os passos abaixo:

  1. Baixar o arquivo nos formatos otf ou ttf
  2. Colar o arquivo da fonte em uma pasta chamada fonts.
  3. Em style use o comando @font-face
  4. Em url insira o nome do arquivo, e em format insira o tipo do formato que podem ser:
    • opentype (otf)
    • truetype (ttf)
    • embedded-opentype
    • truetype-att (Apple Advanced Typography)
    • svg

Configurações de fontes

Para o tamanho das fontes usamos as medidas em ou px
Geralmente o tamanho padrão da fonte é de 1em ou 16px;

Font weight - É o peso da fonte, como se fosse um negrito
Para o peso da fonte podemos usar: lighter, normal, bold ou bolder ou podemos identenficar por números que vão do 100 ao 900.

Shorthands fonts

Para simplificar os comandos podemos utilizar a forma shorthand.
Ao invés de utilizar esse código: 

            h1 {
                font-family: 'Work Sans', sans-serif;
                font-weight: bolder;
                font-size: 3em;
                font-style: italic;
            }
Podemos somente usar esse comando
font: italic bolder 3em 'Work Sans', sans-serif;
Mas necessariamente precisar ser nessa ordem: font-style, font-weight, font-size e font-family.

Variavéis com CSS

  • Crie a marcação :root{}
  • Dentro das chaves insira dois traços + o nome da variavél + :
  • Insira o valor da variavél + ;

Ex1: --cor3: #2fa866;
Ex2: --fonte-padrao: Arial, Verdana, Helvetica, sans-serif;

Para chamar a variavél, use a marcação desejada e depois aperte Ctrl + Espaço desça a barra de rolagem por completo e no final estará a variavél criada ou chame a variavél como no exemplo abaixo:
Ex: font-family: var(--fonte-padrao);
