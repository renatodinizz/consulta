Alguns comandos

Comandos que eu acho necessário

  • padding - Controlam as margens
  • border-radius - Arredonda as margens
  • box-shadow - Constroi uma sombra nas bordas Obs: se colocar os dois primeiros valores zerados a sombra fica dos dois lado
  • text-shadow - Constroi uma sombra no texto
  • text-indent - Para adicionar um espaço no parágrafo
  • font-style - Paracolocar em itálico
  • font-decoration: underline - Para subilinhar um texto
  • margin:auto - Para centralizar uma caixa
  • Transition-duration - Para evidenciar quando o mouse passar em cima
    Ex: transition-duration: 0.5s;
  • line-hight - Para mudar o tamnho do espaçmento entre uma linha e outra
  • text-transform: uppercase; - Para colocar todas as letras em maiusculas
  • font-variant: small-caps; - Para colocar todas as letras em maiusculas, dando destaque para as primeiras letras de cada palavra
  • Cursor: pointer - O ícone do mouse vira uma mãozinha
  • Overflow: hidden - Não deixa o conteúdo passar do limeite

Degradê

Para acrescentar um degradê use o comando background-image: linear-gradient()
Obs: Dentro dos parênteses indique a direção do degradê como: to right, to left, to top, to bottom ou por graus como: 90deg, 45deg
Obs: Depois da direção informe as cores que irão compor o degradê.

Algumas configurações para o degradê:

  • Para fixar o degradê - background-attachment: fixed;
  • Para fazer um degradê circular - radial-gradient: circle; (no lugar de linear-gradient).
  • Para dimensionar quanto de cada cor - Basta coloar a quantidade em porcentagem depois da cor.

Configurações globais da CSS

Para configurar as CSS de maneira global utiliza-se o comando *
Ex: 

            * {
                height: 100%;
            }

Importando fontes

Para importar uma fonte, basta seguir os passos abaixo:

  1. Baixar o arquivo nos formatos otf ou ttf
  2. Colar o arquivo da fonte em uma pasta chamada fonts.
  3. Em style use o comando @font-face
  4. Em url insira o nome do arquivo, e em format insira o tipo do formato que podem ser:
    • opentype (otf)
    • truetype (ttf)
    • embedded-opentype
    • truetype-att (Apple Advanced Typography)
    • svg

Configurações de fontes

Para o tamanho das fontes usamos as medidas em ou px
Geralmente o tamanho padrão da fonte é de 1em ou 16px;

Font weight - É o peso da fonte, como se fosse um negrito
Para o peso da fonte podemos usar: lighter, normal, bold ou bolder ou podemos identenficar por números que vão do 100 ao 900.

Shorthands fonts

Para simplificar os comandos podemos utilizar a forma shorthand.
Ao invés de utilizar esse código: 

            h1 {
                font-family: 'Work Sans', sans-serif;
                font-weight: bolder;
                font-size: 3em;
                font-style: italic;
            }
Podemos somente usar esse comando
font: italic bolder 3em 'Work Sans', sans-serif;
Mas necessariamente precisar ser nessa ordem: font-style, font-weight, font-size e font-family.

Estilizando links

  • text-decoration: none; - Para retirar o subilinhado de um link
  • a:visited {color:red} - Para escolher a cor do link depois de visitado
  • a:hover {color:black} - Para escolher a cor do link quando o mouse passar por cima
  • a:hover {text-decoration: underline} - Para adicionar um subilinhadon ao link quando o mouse passar por cima
  • a:active {color:yellow} - Para adicionar uma cor assim que clicar no link

Variavéis com CSS

  • Crie a marcação :root{}
  • Dentro das chaves insira dois traços + o nome da variavél + :
  • Insira o valor da variavél + ;

Ex1: --cor3: #2fa866;
Ex2: --fonte-padrao: Arial, Verdana, Helvetica, sans-serif;

Para chamar a variavél, use a marcação desejada e depois aperte Ctrl + Espaço desça a barra de rolagem por completo e no final estará a variavél criada ou chame a variavél como no exemplo abaixo:
Ex: font-family: var(--fonte-padrao);
