Iniciando com JavaScript

Alguns comandos simples

Temos alguns comando simples que precisamos guardar.

  • window.alert - É utilizado para exibir uma caixa de mensagem com um alerta na tela.
  • window.confirm - É utilizado para exibir uma caixa de mensagem com uma confirmação na tela.
  • window.prompt - É utilizado para exibir uma caixa de mensagem com um espaço para ser digitado na tela.

Conversão

Para converter uma string em um número inteiro ou flutuante, usamos o comando number.parseInt(n) para converter para um número inteiro e number.parseFloat(n) para converter em um número flutuante.
Obs: Observe que o Int e Float dentro do comando estão com a letra inicial maiúscula, portanto é preciso se manter na norma.

Atualmente no JavaScript podemos simpificar e usar o comando numeber(n) que o próprio JavaScript vai saber se o número é inteiro ou flutuante e converter por ele mesmo, mas isso não descarta o uso dos comandos acima, que servem para ocasiões específicas que eu queira somente um número inteiro ou flutuante.


Para converter algum yipo de variável para uma string usamos o comando n.toString ou string(n) como no exemplo.
string(51) ou 51.toString()

Formatando Strings

Para concatenar uma string com um valor numerico, podemos fazer de duas formas.

Concatenação 'O aluno ' + nome + 'de ' + idade + 'anos, passou na prova com a nota:' + nota
Ou
Template string `O aluno ${nome} de ${idade} anos, passou na prova com a nota ${nota}`

Obs: No segundo exemplo, repare que foi utilizado crase ao invés de aspas.

Alguns outros comando para formatar strings

  • var.lenght - Mostra quantos caracteres tem a string.
  • var.toUpperCase() - Tudo fica em letras MAIUSCULAS.
  • var.toLowCase() - Tudo fica em letras minusculas.

Formatando números

  • var.toFixed(2) - Deixa o número com duas casas decimais, por exemplo: n1 = 1543.5 | n1.toFixed(2) vai me mostrar: 1543.50
  • n1.toLocaleString('pt-BR', {style; 'currency', currency: 'BRL'}) - Esse comando serve para formatar a variavél em uma unidade monetária.
    O resultado seria: R$ 1,543.50
