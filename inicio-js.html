Iniciando com JavaScript

Alguns comandos simples

Temos alguns comando simples que precisamos guardar.

  • .value - É utilizado para atribuir o que foi solicitado como valor, por exemplo, em um formulário onde a pessoa digita um texto, podemos usar o texto digitado como valor.
  • window.alert - É utilizado para exibir uma caixa de mensagem com um alerta na tela.
  • window.confirm - É utilizado para exibir uma caixa de mensagem com uma confirmação na tela.
  • window.prompt - É utilizado para exibir uma caixa de mensagem com um espaço para ser digitado na tela.
  • document.write - É utilizado para escrever algo na página atual.
  • documento.write(p1.InnerText) - É utilizado para escrever algo que está armazenado em uma variável na página atual.
  • documento.write(p1.InnerHtML) - É utilizado para escrever algo que está armazenado em uma variável na página atual, da maneira em que foi formatado em HTML.
  • Hora atual - crie uma variável e chame o comando New Date(), crie outra variável e chame a primeira com o comando getHours variável.getHours, isso serve para ano, dia e outras funções.
  • Inserindo Emoticons - Para adicionar emoticons vamos usar o código de cada meticon específico como : \u{1f449} entre aspas.

Inserir uma imagem

Para inserir uma imagem via JavaScript, criamos uma variável, adicionamos o comando e adicionamos um atributo, que nesse caso é um id.
Seria o mesmo que adicionar o comando img com um id em HTML.
A primeira linha cria uma variável para a imagem e a segunda linha dá um atributo.

Criada a variável é hora de inserir a imagem. Para isso vamos usar um comando já visto, mas com outros atributos.
img.setAttribute('src', 'imagens/adolescente-fem.png') - Chamammos a variável com o comando e dentro dos parenteses usamos o src para puxar a imagem e o restante para identificar em que pasta a imagem está e o nome do arquivo.

Para finalizar e fazer a imagem aparecer vamos usar o comando res.appendChild(img)
res - é a variável criada em JS para onde a imagem vai ser adicionada
appendChild - O comando para fazer a imagem aparecer.
img - A variável que criada em JS para criar a imagem.

Conversão

Para converter uma string em um número inteiro ou flutuante, usamos o comando number.parseInt(n) para converter para um número inteiro e number.parseFloat(n) para converter em um número flutuante.
Obs: Observe que o Int e Float dentro do comando estão com a letra inicial maiúscula, portanto é preciso se manter na norma.

Atualmente no JavaScript podemos simpificar e usar o comando numeber(n) que o próprio JavaScript vai saber se o número é inteiro ou flutuante e converter por ele mesmo, mas isso não descarta o uso dos comandos acima, que servem para ocasiões específicas que eu queira somente um número inteiro ou flutuante.


Para converter algum yipo de variável para uma string usamos o comando n.toString ou string(n) como no exemplo.
string(51) ou 51.toString()

Formatando Strings

Para concatenar uma string com um valor numerico, podemos fazer de duas formas.

Concatenação 'O aluno ' + nome + 'de ' + idade + 'anos, passou na prova com a nota:' + nota
Ou
Template string `O aluno ${nome} de ${idade} anos, passou na prova com a nota ${nota}`

Obs: No segundo exemplo, repare que foi utilizado crase ao invés de aspas.

Alguns outros comando para formatar strings

  • var.lenght - Mostra quantos caracteres tem a string.
  • var.toUpperCase() - Tudo fica em letras MAIUSCULAS.
  • var.toLowerCase() - Tudo fica em letras minusculas.

.toLowerCase - utilizado para converter texto de entrada do usuário em letras minuscúlas, muito útil para não deixar com sensibilidade de maiúscula ou minúscula.

Formatando números

  • var.toFixed(2) - Deixa o número com duas casas decimais, por exemplo: n1 = 1543.5 | n1.toFixed(2) vai me mostrar: 1543.50
  • n1.toLocaleString('pt-BR', {style; 'currency', currency: 'BRL'}) - Esse comando serve para formatar a variavél em uma unidade monetária.
    O resultado seria: R$ 1,543.50
