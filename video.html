Trabalhando com Vídeos

Video inserido no próprio servidor

Vamos usar o comando <Video> e dentro do comando <Video> vamos usar o comando <Source:SRC>
Esse comando é parecido com o IF em JavaScript.
Obs: O comando poster é utilizado para inserir uma thumbnail,
Obs: O comando controls é utilizado para inserir controle no vídeo.

Video inserido de outro servidor

Para inserir um video do youtube vamos seguir alguns passos...

  1. Entre em youtube.com
  2. Escolha o video que deseja
  3. Clique em compartilhar
  4. Clique em incorporar
  5. Copie o código
  6. Cole o código onde desejar
