Tabelas

Inserindo uma tabela simples

Para inserir uma tabela utiliza-se o comando table.
Dentro de table utiliza-se o comando tr "table row" - que é basicamente as linhas da tabela.
Dentro de tr vamos ter o comando td que são os conteúdos dentro da tabela.
Olhe dentro do código, mais fácil do que explicar.
A estilização da tabela eu fiz com CSS.

A1 B1 C1
A2 B2 C2
A3 B3 C3
A4 B4 C4

Pode utilizar o comando vertical-align para centralizar verticalmente as propriedades com os paremetros: top, middle e bottom

Anatomia para tabelas grandes

Para se criar uma tabela grandes, vamos usar a seguinte anatomia.

            
                table 

                    caption 

                    thead
                        tr, td, th 

                    tbody
                        tr, td, th 

                    tfooter
                        tr, td, th

É importante colocar o parametro scope dentro dos títulos (th) para identificar como os dados devem seguir.
Usamos o parametro scope: "col" para indicar que os dados vão seguir a coluna.
Usamos o parametro scope: "row" para indicar que os dados vão seguir a linha
Usamos o parametro scope: "colgorup" para indicar que os dados a seguir terão conteúdo em mais de uma coluna.
Usamos o parametro scope: "rowgroup" para indicar que os dados a seguir terão conteúdo em mais de uma linha.

Efeito Zebrado

Para obter um efeito zebrado podemos usar os seguintes comandos em CSS.

tbody > tr:nth-child () {background: lightgray}

Dentro do parenteses podemos colocar os seguintes parametros...
n2-1 ou odd - Para inserir a cor do background em todas as linhas ímpares
n2 ou even - Para inserir a cor do background em todas as linhas pares

Fixar o Título

Em Css vamos colocar os seguintes comandos.

Primeiro colocamos o position: relative; na tabela.

Depois inserimos o comando em um novo espaço. 

            thead > tr > th {
                position: sticky;
                top:-1px;
                background-color: blue;
            }

position: sticky; - Serve para fixar o título.
top:-1px; - Serve para orientar onde fixar o título.
background-color: blue; - Serve para fixar a cor de fundo junto com o título (deve ser a mesma cor de fundo posto na tabela original).

Mesclar celulas

Para mesclar uma celula em colunas, utiliza-se o comando colspan="número de colunas que deseja mesclar" dentro do td desejado, como no exemplo abaixo.
td colspan="2"

A B
D E F
G H I
J K L

Para mesclar uma celula em linhas, utiliza-se o comando rowspan="número de linhas que deseja mesclar" dentro do td desejado, como no exemplo abaixo.
td rowspan="2"

A B C
D E
H
J K L
