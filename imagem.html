Trabalhando com imagens

Inserir um ícone

Para inserir um ícone utiliza-se o comando Link: favicon na área head
obs: A imagem do ícone deve estar no formato .ico
É recomendado que a imagem do ícone deve estar renomeada como: favicon

Inserir uma imagem

Para inserir uma imagem utiliza-se o comando <img>
Para adicionar uma imagem da internet basta copiar o link da imagem e copia-lo no src do comando.


Imagem flexivél

Vamos aprender como colocar uma imagem flexivél para diferentes tipos de telas, a medida que ampliar a janela a imagem vai se adaptar!

Priemiro usamos o comando <picture>
Dentro do comando <picture> usamos o comando <source:media:type>
Mude o parâmetro de min para max.
Esse comando é parecido com o comando IF em JavaScript, com um detalhe que vai de baixo para cima.
